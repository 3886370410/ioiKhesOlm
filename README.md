# 图书管理系统 java216

## 项目概述

本系统是基于 SpringBoot 和 Vue 的图书管理系统，采用前后端分离的设计模式。系统涵盖用户、图书管理员、管理员三种角色，提供图书借阅、管理等多种功能。

## 技术栈

- 后端：SpringBoot，Shiro，MyBatis-Plus
- 前端：Vue，ElementUI，ECharts
- 数据库：MySQL
- 缓存：Redis
- 开发工具：IDEA，VSCode，Navicat

- 图书推荐
